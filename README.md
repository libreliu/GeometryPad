# GeometryPad

#### Description
A simple Geometry Sketchpad based on C++

#### Software Architecture
Only libc & libc++ required

#### Installation

Besure that you have CMake 2.8 or higher version installed.
> cmake .
> make
> ./GeoPad.elf  # To run the program
